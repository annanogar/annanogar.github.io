## Project

Static portfolio site for Anna Nogaré (annanogar.com). English-only. No framework: Nunjucks -> HTML, SCSS -> CSS, esbuild -> JS. Output: `build/`. Deploy: GitHub Pages via [.github/workflows/deploy.yml](.github/workflows/deploy.yml).

`/it/` pages are redirect stubs only (old Italian URLs -> English). Not real content; ignore them.

Node 26 (nodenv), Yarn 4 (Corepack). Copy `.env.example` -> `.env` once. Build system is **Sonic** (custom, vendored in [sonic/](sonic/)), config in [sonic.config.js](sonic.config.js).

## Commands

- `yarn dev` -- dev server `localhost:8000` + watcher (lints/formats on save)
- `yarn build` -- typecheck + Sonic production build (use over bare `node sonic build`)
- `yarn test` -- build + Playwright a11y suite (zero required-WCAG violations)
- `node sonic lint` -- ESLint + Stylelint

Also: `yarn typecheck` (tsc on `data/*.ts`), `yarn test:a11y` (needs existing `build/`).
Pre-commit gate: lint clean + typecheck + build + test all pass.

## Architecture

**Data**: typed [data/](data/) modules bound to [data/types.ts](data/types.ts) via `satisfies`; composed by [nunjucks.data.js](nunjucks.data.js), exposed to templates as `data.*`. A `Proxy` warns on unknown key access ("Missing data lookups" at exit). Alt text is NOT inline: edit the [data/alts](data/) map (keys like `'projects/foo/01_2x1'`); helpers (`setImageAlts` etc.) resolve strings -> `{ src, alt }` at export time.

**Templates**: [website/templates/pages/](website/templates/pages/) -> URL 1:1, extend `_base.en.html`. Nunjucks filters/globals: [nunjucks.config.js](nunjucks.config.js).

**Components**: [website/components/](website/components/), atomic (`atoms/` -> `molecules/` -> `organisms/` -> `structures/`). Co-located `.html` (Nunjucks macro) + `.scss` + optional `.js`. Component JS auto-loads via glob-import in [main.js](website/assets/scripts/main.js); lifecycle hooks: `init-immediate`, `init-load`, `init-after-load`, `init-delayed-load`.

**Images**: `<name>_<ratio>_<width>w.webp` (e.g. `anna-leaning-forward_1x1_854w.webp`); responsive crops generated by shell scripts in [scripts/](scripts/) (ImageMagick).
